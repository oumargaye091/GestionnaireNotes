# 📝 GestionnaireNotes

Une application Android pour gérer vos notes facilement.

## 📱 Fonctionnalités
- Créer des notes
- Modifier des notes
- Supprimer des notes

## 🛠️ Technologies utilisées
- Java
- Android Studio

## 🚀 Installation
1. Cloner le dépôt
```bash
git clone https://github.com/oumargaye091/GestionnaireNotes.git
```
2. Ouvrir dans Android Studio
3. Lancer sur un émulateur ou appareil Android
